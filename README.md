# AIChatHub

This is a React.js project built using Bolt.AI.

## Setup

1. Clone the repo  
`git clone https://github.com/dnyaneshwarpatange/AIChatHub`

2. Move into folder  
`cd AIChatHub`

3. Install dependencies  
`npm install`  
or  
`yarn install`

4. Run the app  
`npm start`  
or  
`yarn start`

App will run at: http://localhost:3000

## Build for Production

`npm run build`
